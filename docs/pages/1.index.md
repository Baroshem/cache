This module is a Browser & Server Cache middleware for [Nuxt 3](https://v3.nuxtjs.org).

## Features

- Nuxt 3 ready
- Easy Browser Cache Control middleware
- Powerful Server Cache with Redis, memcached, etc [Coming soon]
- TypeScript support
