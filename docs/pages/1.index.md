This module is a wrapper around [Algolia](https://www.algolia.com/) for [Nuxt 3](https://v3.nuxtjs.org).

## Features

- Nuxt 3 ready
- Easy integration with Algolia
- Handy composables like useAlgolia, useSearch, useInitIndex etc
- TypeScript support
